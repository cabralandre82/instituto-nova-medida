# Produto · Instituto Nova Medida

## Visão em uma frase

Plataforma de telessaúde que conecta pacientes brasileiros a médicas
qualificadas para um **tratamento individualizado de emagrecimento** baseado em
ciência, com avaliação online, prescrição quando indicada e acompanhamento
contínuo pelo WhatsApp.

## Tagline

> **Instituto Nova Medida. Não é sobre força de vontade — é sobre o método certo.**

Duplo sentido proposital: "nova medida" como medida do corpo + "nova medida"
como nova abordagem/método.

## Por que existe

A maioria das pessoas que tenta emagrecer falha não por falta de disciplina,
mas porque o **metabolismo, a fome e a saciedade** são governados por
mecanismos fisiológicos que dieta e exercício, isoladamente, não corrigem.
Hoje existem ferramentas farmacológicas que atuam diretamente nesses
mecanismos, mas o acesso a elas é fragmentado, caro e mal orientado.

O Instituto Nova Medida resolve isso oferecendo:
1. **Avaliação médica individual** (online, em plataforma compliant)
2. **Prescrição responsável** quando faz sentido clinicamente
3. **Medicamento manipulado de qualidade** via farmácias auditadas
4. **Acompanhamento contínuo** pelo WhatsApp com a mesma médica

## Modelo de negócio

- **Marketplace de médicas (PJ contratadas)** — o instituto é a marca,
  as médicas atendem
- **Plano por ciclo de 90 dias** — alinhado ao ciclo clínico de avaliação
- **Consulta gratuita se não houver indicação** — remove fricção, alinha
  incentivos, gera confiança
- **Pagamento upfront** (PIX/boleto à vista ou cartão em 3x sem juros)
- **Compra do medicamento em plataforma separada** (instituto não fatura
  medicamento; farmácia recebe direto)
- **Split automático para médica** via Asaas

## Jornada do paciente

```
Anúncio (Meta/Google) ou indicação
  → Landing → Quiz (4 perguntas) → Captura nome+WhatsApp
  → Mensagem WhatsApp 1 (boas-vindas)
  → MSG 2-6 (educação, identificação, custo)
  → Marca consulta (fila ou agendamento)
  → Anamnese curta pré-consulta (link)
  → TCLE eletrônico
  → Teleconsulta (vídeo, plataforma própria)
    ├─ Aprovado → Receita Memed → Pagamento Asaas → Pedido na farmácia parceira
    │              → Acompanhamento WhatsApp (semanal automatizado + médica)
    │              → Reconsulta em 30 dias → Renovação ou ajuste de dose
    └─ Não aprovado → Consulta gratuita → Encaminhamento (ex: nutricionista)
```

## Diferenciais competitivos

1. **Consulta gratuita se não houver indicação** — Voy, Eva e similares
   cobram avaliação. Reduz fricção e gera confiança.
2. **Mesma médica acompanha o paciente** — não é "atendimento por
   plantão". Vínculo terapêutico real.
3. **WhatsApp como canal principal de acompanhamento** — onde o paciente
   já está, sem precisar instalar app.
4. **Tiers de plano** preparados para retatrutida (próxima geração).
5. **Preço competitivo** com farmácia de manipulação auditada.

## Métricas de sucesso (North Star)

- **MRR** (receita recorrente mensal)
- **% renovação ciclo 2** (retenção real)
- **CAC payback** (quanto tempo para recuperar custo de aquisição)
- **NPS pós-1ª dose** (qualidade percebida)
